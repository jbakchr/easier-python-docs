# easier-python-docs

> Find Python solutions by problem, not by module.

The official Python documentation is excellent.

However, it is primarily designed as a reference.

This site helps bridge the gap between:

```text
I have a problem
```

and

```
I found the right Python documentation
```

## Categories

### Filesystem

- Join paths
- Read a file
- Write a file
- Check if a file exists

### Strings

- Split strings
- Join strings
- Replace text

### JSON

- Read JSON
- Write JSON

## Philosophy

This site does not replace the official Python documentation.

Instead, it helps you arrive at the correct documentation faster.

Every page includes:

- Problem description
- Recommended Pythonic solution
- Alternatives
- Links to official Python documentation
